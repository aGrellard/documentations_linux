Copier un fichier sur Linux avec un terminal
Vous pouvez aussi copier ou déplacer des fichiers depuis un terminal.
Soit donc en ligne de commandes.
L'avantage ici est que cela fonctionne sur toutes les distributions Linux.
Pour cela, plusieurs commandes sont utiles :
    cd : pour changer de répertoire. Par exemple : cd /home/malekalmorte
    ls : pour lister le contenu d'un dossier
    cp : copier les fichiers
    mv : déplacer des fichiers

Ainsi on utilise cd pour se rendre dans le répertoire source.
Il faut donc changer les répertoires selon votre configuration.
Par exemple :
	cd /home/malekalmorte/Documents
	
Vous pouvez visualiser le contenu du dossier avec la commande ls
Si on utilise la commande seule, vous avez les fichiers en colonne.
Pour obtenir la liste avec les détails, il faut utiliser l'option -v :
	ls -l	
	
Ensuite on utilise la commande cp pour copier les fichiers.
La syntaxe est :
	cp source destination	
Par défaut, cp n'affiche pas les fichiers lors de la copie.
Pour forcer l'affichage, il faut utiliser le paramètre -v.
Soit donc :
	cp -v source destination
	
Pour copier tout le contenu d'un dossier, il faut utiliser en source le caractère *
Ce qui donne :
	cp * /destination/	
	
Pour copier un seul fichier, on met ce dernier en source.
Par exemple :
	cp nomdufichier /destination
	
Pour déplacer les fichiers, il faut utiliser la commande mv.
La syntaxe est identique à celle de cp.
	mv source destination

Modifier les permissions d'un fichier ou d'un dossier
Pour modifier les permissions d'un fichier ou d'un dossier

$ sudo chmod 777 ./nom_du_fichier
modifie les permissions pour le fichier

$ sudo chmod 777 ./nom_du_dossier
modifie les permissions pour le dossier

$ sudo chmod -R 777 ./nom_du_dossier
modifie les permissions pour le dossier et tous les fichiers du dossier

Les permission sont données dans l'ordre user - groupe - autre
	La lecture = 4
	L'écriture = 2
	L'exécution = 1
	Tous les droits = 7
